Background

Dr. Mariappan Jawaharlal is the Associate Dean of the College of Engineering and Computer Science at California State University, Sacramento. He has more than 25 years of academic, industrial and entrepreneurial experience in engineering and is renowned for his innovative, engaging teaching pedagogy. As someone who is interested in stock trading Dr Jawaharlal would like our team to create an application that can help to optimally analyse short term stock trades.

Scope

Scope Overview

Dr. Jawaharlal would like to create a smart (and possibly intelligent) application that can help to efficiently track stock trades. This data will then be used to identify patterns which can help the client consolidate all stock data into one workable application. The application is expected to be used for personal use initially, with the potential to commercialize this site for all stock traders interested in short term trading.


Which parts of the business problem will be addressed?

Dr. Jawaharlal is interested in the developement of a website to analyse stock trades. The application will consolidate all processes and tools into one workable application that will give the user the ability to help analyse historical data for increasing profitable trades over time.


What are the project goals?

  • An application capable of tracking trades and help identify stock trends
  • Track Stock Trades
    • Manual input of stocks or the option to modify (past) input
    • Overview (summary) of how a trader is doing each month month
    • View a report of trades based on manual input or file input
    • Sort report by various columns, parameters, or calculations
  • Pattern Identifcation
    • Perform calculations used to identify potential stock patterns
    • Create new strategies based on various criteria(s)
    • Add patterns to specific stock trades
    • Extra: Automatically getermine if a trade fits currently defined criteria(s)
    • Extra: Automatically generate new patterns
  • Export reports to a CSV File
  • Inport Files From Brokerages
    • Extra feature. All Brokerages export data with different formats
  • News Page
    • Extra feature. Not needed but may be useful to quickly access news related to market activity
  • All users will have individual accounts
    • To Begin, We Will have:
      • One Account For Client To Begin
      • One Account For Developement Team
      • Potential Commercialization of the product with multiple users
